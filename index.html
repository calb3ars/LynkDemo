Features

  • Real-time Uber and Lyft Prices

    Lynk uses the Uber and Lyft APIs to provide real-time fare estimates based on your pickup location and destination

  • Uber vs. Lyft

    Ride types are compared side-by-side so you can easily compare and choose your ride based on your needs (price, time, # passengers)

  • Book your Fare

    Once you've decided on a fare, Lynk helps you pre-populate the pickup and destination information in the respective Uber or Lyft app so you can quickly book and get going!

Technologies

The following languages and APIs were used in the making of Lynk

  • React Native
  • Google Maps
  • Geocoding
  • Uber API
  • Lyft API

React Native was used as the foundation for creating Lynk on the Apple iOS platform. Google Maps API was used to find user current location and destination addresses. The results were rendered using React Native Maps. For our API calls to Uber and Lyft's APIs, the Google Maps locations were translated into longitude and latitude coordinates using the React Native Geocoding API. Fare estimates and links to the Uber and Lyft applications used the Uber and Lyft APIs, respectively.
