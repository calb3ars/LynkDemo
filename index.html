Welcome to Lynk!

    Lynk is a mobile app built using React Native for the Apple iOS platform.
    For the price conscious, comparing transportation costs between Lyft and Uber requires you to open both apps to check. This app will allow users to do a price comparison from within one single app.
    Explore the different features listed below:
  • Ability to input either an address or place name
  • Places a marker at the pickup location and destination once there is an input
  • Ability to select the number of passengers
  • Filter ride costs by price and ride time
  • Ability to select a ride and redirect to associated app with pre-populated data
  • Centers map view on user's current location
  • Pick up location defaults to user's current location
